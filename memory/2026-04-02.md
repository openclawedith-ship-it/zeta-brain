### Key Design Principles
- **Zero cloud, zero external dependencies for core functionality** — every tool has Python fallback
- **Graceful degradation** — if nmap is missing, Python socket scanning works; if gobuster missing, Python concurrent requests work; if sqlmap missing, Python SQLi heuristics work
- **Automatic tool detection** — `_has_tool()` checks before every external tool invocation
- **SQLite persistence** — all findings stored locally for longitudinal analysis
- **AI-assisted reporting** — uses local Qwen2.5 model, falls back to templates
